# Code Of Conduct

This project follows a simple contributor standard:

- Be respectful and direct.
- Assume good intent, but prioritize maintainable code and user safety.
- Keep discussion focused on the problem, reproduction, and fix.
- Do not post secrets, printer access codes, private network details, or API keys.
- Do not harass, insult, or target contributors.

Maintainers may edit, hide, or remove comments and issues that violate these
rules. Repeated abuse can result in being blocked from the project.
